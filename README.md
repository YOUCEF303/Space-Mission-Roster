Space Mission Roster 🚀
A JavaScript-based management system designed for space mission control to organize, validate, filter, and structure astronaut crews for upcoming space missions.

This project was built to practice core JavaScript programming concepts, data manipulation, and algorithmic thinking.

🛠️ Features & Concepts Implemented
Data Validation: Ensures crew indices and chunk sizes meet mission parameters before execution.

Array Manipulation (slice & splice): Safely swapping astronaut positions within a squad without mutating the original data.

Bubble Sort Algorithm: Implementing a custom descending sorting algorithm to prioritize astronauts based on their priority level.

Filtering & Optimization: Extracting EVA-eligible astronauts dynamically using conditional looping.

2D Array Chunking & Nested Loops: Grouping the final selected crew into smaller, organized mission teams (Chunks) and logging them seamlessly.

💻 Tech Stack
Language: JavaScript (ES6+)
